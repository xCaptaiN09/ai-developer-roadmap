You are building AI-powered products — apps where users interact with AI you've assembled from existing models. Think: upload a PDF and ask questions, a smart chatbot for a business, an AI tool that automates a task. You don't need to build the AI brain. You use it as a component and build the product around it. No heavy math required. This is Path A (AI App Developer) as primary, with just enough deployment skills (Docker + GCP) to ship your projects live. A product that only runs on your laptop is not a product. GitHub portfolio + deployed projects = your degree replacement.

// Phases — follow in order
PHASE 0
Foundation
Now → May 2025  ·  ~2 months  ·  4th Sem parallel
Python — Intermediate
  • 100 Days of Code: The Complete Python Pro Bootcamp — Dr. Angela Yu — Udemy — ~₹499 on sale — 4–6 weeks
  • Link: udemy.com → search "100 Days of Code Angela Yu"
  • Project-based. You build something every day. Modern, clean, industry-style Python.
  • After completing → do Kaggle Python Certificate at kaggle.com/learn/python — free, credible
SQL Basics
  • CS50's Introduction to Databases with SQL — Harvard — completely free — certificate included — 2 weeks
  • Link: cs50.harvard.edu/sql
  • Harvard certificate. Highly credible for college and recruiters.
Git + GitHub
  • Git and GitHub for Beginners — freeCodeCamp — YouTube — free — 1 hour — 1 week
  • Search on YouTube: "Git and GitHub for Beginners freeCodeCamp"
  • No certificate needed. Your GitHub commit history IS the proof.
  • Start pushing every script from day one. Even small ones.
Outcome: You stop being a beginner. Every phase after this becomes learnable.
PHASE 1
DSA — Interview Survival
June 2025 → Nov 2025  ·  ~6 months  ·  5th Sem parallel
Strategy — Do NOT rush this
  • Month 1–2: Arrays, strings, hashmaps — Easy problems only (NeetCode 150)
  • Month 3–4: Sliding window, two pointers, binary search, linked lists
  • Month 5–6: Trees, recursion, basic dynamic programming, graphs intro
  • Target: 80–100 Easy + 40–50 Medium by end. Not 100 Mediums cold.
  • 1 problem on college days, 2–3 on Sundays. Python only.
Resource
  • NeetCode.io Roadmap — free — primary resource
  • Link: neetcode.io — roadmap + video explanations for every problem
  • YouTube: youtube.com/@NeetCode — watch his explanation when stuck
  • LeetCode — practice platform — leetcode.com — free tier is enough
Outcome: You pass screening rounds. DSA is the filter most students fail at.

Aug 2025 → Nov 2025  ·  ~3 months  ·  Runs parallel with Phase 1

  • Search on YouTube: "FastAPI Course for Beginners freeCodeCamp"

LLMs + Prompting

RAG Systems

  • Link: deeplearning.ai/short-courses — use this, NOT old Udemy courses
  • LangChain official docs for reference: python.langchain.com
  • LlamaIndex Course — free — certificate — 1–2 weeks
  • Link: docs.llamaindex.ai
Vector Databases
  • FAISS — official docs + hands-on — 1 week — faiss.ai
  • Pinecone Academy — free — certificate — 1 week — docs.pinecone.io
Local LLMs
  • Ollama — hands-on — 1 week — ollama.com
  • Run Mistral or Llama3 locally on your laptop. 16GB RAM handles it.
  • No course needed. Just install and experiment.
Outcome: You can build real RAG applications end-to-end. This is your primary job skill.
