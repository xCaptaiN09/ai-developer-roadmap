You are building AI-powered products — apps where users interact with AI you've assembled from existing models. Think: upload a PDF and ask questions, a smart chatbot for a business, an AI tool that automates a task. You don't need to build the AI brain. You use it as a component and build the product around it. No heavy math required. This is Path A (AI App Developer) as primary, with just enough deployment skills (Docker + GCP) to ship your projects live. A product that only runs on your laptop is not a product. GitHub portfolio + deployed projects = your degree replacement.

// Phases — follow in order
PHASE 0
Foundation
Now → May 2025  ·  ~2 months  ·  4th Sem parallel
Python — Intermediate
  • 100 Days of Code: The Complete Python Pro Bootcamp — Dr. Angela Yu — Udemy — ~₹499 on sale — 4–6 weeks
  • Link: udemy.com → search "100 Days of Code Angela Yu"
  • Project-based. You build something every day. Modern, clean, industry-style Python.
  • After completing → do Kaggle Python Certificate at kaggle.com/learn/python — free, credible
SQL Basics
  • CS50's Introduction to Databases with SQL — Harvard — completely free — certificate included — 2 weeks
  • Link: cs50.harvard.edu/sql
  • Harvard certificate. Highly credible for college and recruiters.
Git + GitHub
  • Git and GitHub for Beginners — freeCodeCamp — YouTube — free — 1 hour — 1 week
  • Search on YouTube: "Git and GitHub for Beginners freeCodeCamp"
  • No certificate needed. Your GitHub commit history IS the proof.
  • Start pushing every script from day one. Even small ones.
Outcome: You stop being a beginner. Every phase after this becomes learnable.
PHASE 1
DSA — Interview Survival
June 2025 → Nov 2025  ·  ~6 months  ·  5th Sem parallel
Strategy — Do NOT rush this
  • Month 1–2: Arrays, strings, hashmaps — Easy problems only (NeetCode 150)
  • Month 3–4: Sliding window, two pointers, binary search, linked lists
  • Month 5–6: Trees, recursion, basic dynamic programming, graphs intro
  • Target: 80–100 Easy + 40–50 Medium by end. Not 100 Mediums cold.
  • 1 problem on college days, 2–3 on Sundays. Python only.
Resource
  • NeetCode.io Roadmap — free — primary resource
  • Link: neetcode.io — roadmap + video explanations for every problem
  • YouTube: youtube.com/@NeetCode — watch his explanation when stuck
  • LeetCode — practice platform — leetcode.com — free tier is enough
Outcome: You pass screening rounds. DSA is the filter most students fail at.
PHASE 2
Applied GenAI — Core Skill
Aug 2025 → Nov 2025  ·  ~3 months  ·  Runs parallel with Phase 1
FastAPI
  • FastAPI Course for Beginners — freeCodeCamp — YouTube — free — 2–3 weeks
  • Search on YouTube: "FastAPI Course for Beginners freeCodeCamp"
  • Official docs for reference: fastapi.tiangolo.com
  • Build one small project: a working REST API with endpoints
LLMs + Prompting
  • ChatGPT Prompt Engineering for Developers — DeepLearning.AI — free — certificate — 1 week
  • Link: deeplearning.ai/short-courses
  • Understand how to talk to LLMs programmatically, not just in chat
RAG Systems
  • LangChain for LLM Application Development — DeepLearning.AI — free — certificate — 3–4 weeks
  • Link: deeplearning.ai/short-courses — use this, NOT old Udemy courses
  • LangChain official docs for reference: python.langchain.com
  • LlamaIndex Course — free — certificate — 1–2 weeks
  • Link: docs.llamaindex.ai
Vector Databases
  • FAISS — official docs + hands-on — 1 week — faiss.ai
  • Pinecone Academy — free — certificate — 1 week — docs.pinecone.io
Local LLMs
  • Ollama — hands-on — 1 week — ollama.com
  • Run Mistral or Llama3 locally on your laptop. 16GB RAM handles it.
  • No course needed. Just install and experiment.
Outcome: You can build real RAG applications end-to-end. This is your primary job skill.
PHASE 3
ML Theory + PyTorch — Just Enough
Dec 2025 → Feb 2026  ·  ~3 months  ·  6th Sem
PyTorch First — then Theory
  • PyTorch Fundamentals — DeepLearning.AI — free — certificate — 2–3 weeks
  • Link: deeplearning.ai/short-courses
  • Do this BEFORE Andrew Ng. Learn the tool first, then the theory.
  • Machine Learning Specialization — Andrew Ng — Coursera — certificate — 8 weeks
  • Link: coursera.org → search "Machine Learning Specialization Andrew Ng"
  • Apply for financial aid on Coursera — almost always approved — makes it free
Math Rule
  • Understand what gradient descent does — not the full derivation
  • Understand backprop intuitively — not mathematically
  • That's enough. You're a builder, not a researcher.
Outcome: You understand models well enough to debug and improve them. You can talk intelligently in interviews about how AI works.
PHASE 4
Deployment — Make It Real
Mar 2026 → Jun 2026  ·  ~3 months  ·  6th–7th Sem
Docker (mandatory) — This is Path A deployment, not pure Path B
  • Docker and Kubernetes: The Complete Guide — Stephen Grider — Udemy — ~₹499 on sale — certificate — 3–4 weeks
  • Link: udemy.com → search "Docker and Kubernetes Complete Guide Stephen Grider"
  • Do Docker section first. Kubernetes later if time permits.
  • Containerize every project you build from here on.
Cloud — GCP Vertex AI (not AWS)
  • Google Cloud Skills Boost — free + paid — Google badges + certificate — 4–6 weeks
  • Link: cloudskillsboost.google
  • After completing → attempt GCP Associate Cloud Engineer exam — real Google certificate, strong in India
  • GCP over AWS for India AI/ML job market. Vertex AI pairs with Gemini APIs.
Monitoring
  • Evidently AI — free — docs + YouTube — 1–2 weeks
  • Link: evidentlyai.com
  • Model drift, data drift. Know how to tell when your AI is going wrong.
Parallel Reading
  • Designing ML Systems — Chip Huyen — book — free chapters online — read 3–4 chapters
  • Link: huyenchip.com/book
  • Teaches you how to explain your architecture, tradeoffs, scaling decisions in interviews
Outcome: You can deploy, version, and monitor models in production. This separates you from 90% of AI&DS graduates.
PHASE 5
Flagship Project + Portfolio
Starts Phase 2, completes by Jul 2026  ·  Ongoing
Mandatory Flagship Project — End-to-End RAG App
  • FastAPI backend
  • Vector DB (FAISS or Pinecone)
  • LLM — API (OpenAI/Gemini) + local Ollama fallback
  • Dockerized and deployable
  • Real use case — pick one: legal docs, medical reports, student Q&A, business FAQ
Supporting Portfolio Work
  • Every script, every project → GitHub with clean README
  • LinkedIn: post about what you built, what broke, how you fixed it
  • 2–3 smaller projects across different domains (not just one RAG app)
  • Kaggle Learn tracks: ML, Python, SQL — free certificates, credible
Outcome: Portfolio replaces your college brand. This is what gets you interviews.

Non-Negotiable Rules
  • 01Every single thing you build goes on GitHub. Even Phase 0 practice scripts. Recruiters from tier-2 colleges are filtered by GitHub activity before resume content.
  • 02LinkedIn presence matters as much as GitHub. Post about what you built, what broke, and how you fixed it. That gets recruiter attention. Polished READMEs alone don't.
  • 03Do not skip Phase 0. Students always want to jump to the exciting stuff. The Python and SQL gaps will kill you in technical interviews.
  • 04DSA is not optional. It feels unrelated to AI work. It isn't. It's the filter round. Most AI&DS students skip it and fail screening.
  • 05College marks are secondary. Minimum passing is fine. Time spent chasing marks beyond that is time stolen from your portfolio.
  • 06Consistency beats intensity. 12 hours a week every week beats 40 hours one week then nothing. This is a 22-month plan, not a sprint.
